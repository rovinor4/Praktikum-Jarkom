# Daftar Gambar dan Instruksi Modul 11 DHCP

| File | Isi Screenshot | Instruksi |
| --- | --- | --- |
| `asset/01-dhcp-dora.avif` | Semua paket DHCP | Gunakan filter `dhcp` atau `bootp`, lalu screenshot paket Discover, Offer, Request, dan ACK. |
| `asset/02-dhcp-discover.avif` | Detail DHCP Discover | Pilih paket Discover, buka bagian DHCP, lalu screenshot. |
| `asset/03-dhcp-offer.avif` | Detail DHCP Offer | Pilih paket Offer, buka bagian DHCP Options, lalu screenshot. |
| `asset/04-dhcp-request.avif` | Detail DHCP Request | Pilih paket Request, tampilkan requested IP address. |
| `asset/05-dhcp-ack.avif` | Detail DHCP ACK | Pilih paket ACK, tampilkan lease time, router, dan DNS server. |

Catatan: Pada MacBook, ganti `en0` jika interface aktif berbeda, misalnya `en1`.
